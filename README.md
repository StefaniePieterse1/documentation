# documentation

Holds templates for technical documentation.

## Available Templates

| Template | Description |
|---|---|
| [TECHNICAL_DOCUMENTATION_TEMPLATE.md](./TECHNICAL_DOCUMENTATION_TEMPLATE.md) | General-purpose technical documentation template covering stakeholders, project benefits, data tables, data quality, usage, security, testing, deployment, and more. |

## How to Use

1. Copy the relevant template file into your project repository.
2. Rename it to suit your project (e.g., `MY_PROJECT_DOCUMENTATION.md`).
3. Replace every placeholder in square brackets `[ ]` with your project-specific content.
4. Remove any sections that are not applicable to your project.
